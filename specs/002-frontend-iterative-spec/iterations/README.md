# Iterations Folder Guide

This folder stores cycle execution records for feature 002.

## Naming Convention

- Use file name format: iteration-XXX.md
- Example: iteration-001.md

## Required Sections Per Iteration File

1. Iteration metadata
2. Scope boundary declaration
3. Red-green evidence references
4. Coverage gate summary
5. Integration boundary notes
6. Delivery summary with changed paths

## Source Templates

- checklists/iteration-slice-template.md
- checklists/red-green-evidence-template.md
- checklists/coverage-gate-template.md
- checklists/integration-boundary-template.md

## Close Criteria

An iteration can be considered closed only when:

- scope boundary policy is respected,
- red-green evidence is present,
- coverage policy passes,
- progress tracker is updated.
